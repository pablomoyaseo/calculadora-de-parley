Calculadora de Parley Tradicional

Tecnologias Utilizadas en este Proyecto:

HTML

  • Elementos de Formularios (fieldset, legend, label, input, button)
  • Uso de Google Fonts
  • Uso de Etiquetas de Maquetado (Header, Main, Section, Footer).
  • Uso de etiqueta Picture para incluir formato de imagen de nueva Generación (AVIF y WEBP).

CSS

  • Normalize.
  • SCSS.
  • Use and Forward (SCSS). Nueva forma de importar hojas de Estilos en SASS Y SCSS.
  • Uso de gradients.
  • Uso de Mixins y variables con SCSS.
  • CSS Grid y Flexbox.
  • Estilos a prompt y alert de Javscript.

Node JS

  • Gulp File.
  • Compilar SCSS.
  • Convertir a CSS.
  • Utilizar Autoprefixer.
  • Minificar CSS.
  • Mejor tamaño de imagenes.
  • Convertir imagenes a WEBP y a AVIF.

JavaScript

  • Modificar el DOM (Crear Elemento, agregar y remover clases.
  • Definicion de Variables.
  • Uso de Condicionales (If, Else).
  • Operadores de Comparación y Operadores Lógicos.
  • Uso de Funciones.
  • Escuchar Eventos.
  • Mejoramos y simplificamos el código de la Calculadora anterior reduciendo el código HTML y el Código Javascript. Pasamos de 225 lineas de código JS a 113.
  • Agregamos un elemento de Entrada en donde el usuario coloca la cantidad de juego a entrar y se crean los inputs en el formulario de acuerdo a la cantidad indicada. La limitamos a 20 para evitar coloquen valores muy altos o ciclos infinitos.
  • Utilizamos ciclos for, ciclos foreach, almacenamos valores en un Array y luego recorrimos con un ciclo foreach.
  • Reducimos la cantidad de variables creadas.
  • Uso de Elementos de Entrada para el Usuario (prompt) y elementos de salida (alert).
